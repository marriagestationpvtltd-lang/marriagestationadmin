Convert msfinal Android/iOS app to Flutter Web

## Summary
Complete conversion of Marriage Station mobile app to web application
with responsive design, platform-agnostic services, and web optimization.

- ✅ Web platform support with clean URL routing

- ✅ Web-optimized UI with hover effects
- ✅ PWA support with manifest
- ✅ Firebase Hosting configuration

## Build & Deploy
```bash
# Install dependencies
cd msfinal && flutter pub get

# Run locally
flutter run -d chrome

# Build for production
./build_web.sh

# Deploy to Firebase
firebase deploy --only hosting
```
